# hummbl-mcp-utf

MCP server exposing the HUMMBL Unified Tier Framework: problem classification, model recommendation, tier assessment, and framework overview via stdio JSON-RPC.

## Install

```bash
pip install hummbl-mcp-utf
```

## Run

```bash
hummbl-mcp-utf
```

## Dependencies

None (stdlib only).
